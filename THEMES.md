# 🎨 Temas & Paletas de Cores

Aqui estão alguns temas alternativos que você pode usar no portfolio.

## 📋 Como Usar um Tema

1. Abra `style.css`
2. Localize a seção `:root`
3. Copie os valores de cores do tema desejado
4. Cole sobre as cores atuais

## 🛠️ Como Criar Seu Próprio Tema

1. **Escolha uma cor base** (fundo)
2. **Determine cores claras** (texto e destaques)
3. **Mantenha contraste** mínimo de 4.5:1 para acessibilidade
4. **Teste em diferentes dispositivos**

### Ferramenta de Paleta Online

- [Coolors.co](https://coolors.co/) - Gerar paletas aleatórias
- [Adobe Color](https://color.adobe.com/) - Schemas profissionais
- [Contrast Checker](https://webaim.org/resources/contrastchecker/) - Validar contraste

## 🌈 Aplicar Tema Cíclico (JS)

Adicione este script ao `script.js` para permitir trocar temas:

```javascript
const themes = [
    {
        name: 'dark-terminal',
        colors: {
            '--color-accent': '#8b5cf6',
            '--color-primary': '#e6e6e6',
            '--color-bg-primary': '#0d0d0d'
        }
    },
    {
        name: 'blue-void',
        colors: {
            '--color-accent': '#06b6d4',
            '--color-primary': '#e0e6ff',
            '--color-bg-primary': '#0a0e27'
        }
    }
];

function toggleTheme() {
    const root = document.documentElement;
    const currentTheme = localStorage.getItem('theme') || 0;
    const nextTheme = (parseInt(currentTheme) + 1) % themes.length;

    const colors = themes[nextTheme].colors;
    Object.keys(colors).forEach(key => {
        root.style.setProperty(key, colors[key]);
    });

    localStorage.setItem('theme', nextTheme);
    console.log('Tema alterado para:', themes[nextTheme].name);
}

// Pressione Ctrl + T para trocar temas
document.addEventListener('keydown', (e) => {
    if ((e.ctrlKey || e.metaKey) && e.key === 't') {
        e.preventDefault();
        toggleTheme();
    }
});
```

## 📱 Temas Responsivos

Para aplicar temas diferentes baseado no horário:

```javascript
function setThemeByTime() {
    const hour = new Date().getHours();
    const root = document.documentElement;

    if (hour >= 6 && hour < 18) {
        // Tema claro durante dia
        root.style.setProperty('--color-bg-primary', '#f5f5f5');
        root.style.setProperty('--color-text-primary', '#1a1a1a');
    } else {
        // Tema escuro durante noite
        root.style.setProperty('--color-bg-primary', '#0d0d0d');
        root.style.setProperty('--color-text-primary', '#e6e6e6');
    }
}

// Executar ao carregar
setThemeByTime();
```

- [ ] Contraste mínimo 4.5:1 entre texto e fundo

- [ ] Não use apenas cor para indicar estado
- [ ] Cite a paleta de cores em alt text de imagens
